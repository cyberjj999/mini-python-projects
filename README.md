# mini-python-projects
This repository contains the source code for my mini python projects

github_webscrap.py - webscrap data from github based on user's input username <br />
file_rename.py - renames all the files in a certain directory to your desired name <br />
get_weather.py - retrieves weather from open weather api and display the city's weather to the user <br />
countdown.py - countdown x seconds (input by user) and prints "Time is Up" when the countdown is over <br />
password_generator.py - generate x number of random password with y length <br />
qr_code_genreator.py - generate qr code from url and download qr code in .eps and .svg format
